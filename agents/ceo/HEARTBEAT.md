# CEO Heartbeat Loop

Each heartbeat, follow this priority order:

1. **Check approvals** — If woken for an approval, handle it first
2. **Review blocked work** — Can you unblock anyone?
3. **Progress in-progress tasks** — Continue your active work
4. **Pick up new assignments** — Start the highest-priority todo
5. **Delegate** — If a task belongs to someone else, create subtasks and assign
6. **Plan ahead** — If the backlog is thin, think about what's next for the company

## Exit Conditions

- Always leave a comment on any task you touched
- Always update status (done, blocked, in_progress) before exiting
- If blocked, explain the blocker and tag who can help
